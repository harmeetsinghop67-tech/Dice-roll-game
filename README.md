🎲 Dice Roll Game (Python)

A simple and fun Python-based Dice Roll Game where the user plays against the computer. The game uses random number generation to simulate dice rolls and determines the winner after each round.

🚀 Features
🎮 Menu-driven interface
🎲 Random dice roll using random module
🧠 Simple game logic using functions
🏆 Displays winner after each round
🔁 Play multiple rounds until exit
🛠️ Technologies Used
Python 3
random module

📂 Project Structure
dice_game.py   # Main game file
README.md      # Project documentation

▶️ How to Run
Make sure Python is installed
Download or clone the repository
Open terminal/command prompt
Run the file:
python dice_game.py

🎯 How to Play
Choose Play Game from the menu
Press Enter to roll the dice
Both user and computer roll a number (1–6)
The higher number wins 🎉
Continue playing or exit
